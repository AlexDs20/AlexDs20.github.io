EasyKindle

What and why?

The internet is full of very interesting articles to read. However one easily loses focus and starts doing something else on the tablet/computer. On a phone, I find it nearly impossible to just read an article.

My solution to this problem was to make a firefox extension. The extension allows you to select parts of a webpage and save it as an html file (including images). That downloaded file can then simply be send to your kindle as for any other book / file you send to your kindle.
Enjoy your reading on your kindle!

Where?

The code for the extension is working and available on github.
